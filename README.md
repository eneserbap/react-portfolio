# React Portfolio

Bu proje, öğrenci ve yeni başlayan yazılım geliştiriciler için hazırlanmış modern ve sade bir React portföy şablonudur. Kendi bilgilerinizi ve projelerinizi kolayca ekleyebileceğiniz şekilde tasarlanmıştır.

## Kurulum

Projeyi bilgisayarınıza klonlayın ve gerekli paketleri yükleyin:

```bash
git clone https://github.com/eneserbap/react-portfolio.git
cd react-portfolio
npm install
npm start
```

Tarayıcınızda `http://localhost:3000` adresinden projeyi görüntüleyebilirsiniz.

## Özelleştirme

Projede yer alan içerikleri ve bilgileri doğrudan `src` klasörü içindeki component’leri düzenleyerek değiştirebilirsiniz.

* Hero bölümü
* Projeler
* Footer ve sosyal medya linkleri

## Deploy

Projeyi canlıya almak için Vercel veya benzeri bir servis kullanabilirsiniz. Build komutu:

```bash
npm run build
```

Output klasörü: `build`

Şablonu canlı olarak görüntülemek için:  
[https://reactportfolioapp004.vercel.app](https://reactportfolioapp004.vercel.app)

Aşağıdaki Vercel butonuna tıklayarak projenizi Vercel uygulamasına tek tuşla deploy edebilirsiniz.
